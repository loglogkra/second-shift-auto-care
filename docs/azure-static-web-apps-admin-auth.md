# Azure Static Web Apps admin authentication

This app uses Azure Static Web Apps built-in Microsoft Entra authentication for the admin area. It does not implement a custom username/password flow and does not store admin passwords.

## Configuration location

`staticwebapp.config.json` belongs in `src/SecondShiftAutoCare.Client/staticwebapp.config.json`. That is the Blazor WebAssembly app's output root, so Azure Static Web Apps can copy the file to the deployed site root.

## Protected routes

- `/admin/login` redirects to `/.auth/login/aad?post_login_redirect_uri=/admin/dashboard`.
- `/admin/logout` redirects to `/.auth/logout?post_logout_redirect_uri=/`.
- `/admin/*` requires the `admin` role.
- `/api/admin/*` requires the `admin` role.
- HTTP 401 responses redirect to `/admin/login`.

The customer-facing service request form and public `POST /api/service-requests` submission endpoint remain public.

## Assign your Microsoft account to the admin role

1. Open the Azure portal.
2. Go to your Static Web App resource for Second Shift Auto Care.
3. Select **Role Management**.
4. Choose **Invite**.
5. Select the Microsoft Entra / Microsoft identity provider.
6. Enter your Microsoft account email address.
7. Add the role name `admin`.
8. Create the invitation and open the generated invitation link while signed into that Microsoft account.
9. After accepting the invitation, browse to `/admin/login` and Azure Static Web Apps will send you to Microsoft Entra sign-in, then back to `/admin/dashboard`.

## Local testing notes

Azure Static Web Apps authentication endpoints such as `/.auth/login/aad`, `/.auth/logout`, and `/.auth/me` are provided by Azure Static Web Apps. They are not fully available when running only the Blazor dev server and Azure Functions host directly.

For end-to-end auth testing, deploy to Azure Static Web Apps or use the Azure Static Web Apps CLI emulator. API authorization should still be validated in Azure because production requests include the `x-ms-client-principal` header generated by Static Web Apps.
